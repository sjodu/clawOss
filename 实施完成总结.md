## 已知问题与解决方案

### 问题 1: TypeScript 类型错误
- **现象**: `dashboard-reporter/handler.ts` 显示 `process` 未定义
- **影响**: 仅编辑器警告，运行时无影响
- **解决**: 可忽略，或安装 `@types/node`

### 问题 2: bc 命令不存在
- **现象**: `budget-status.sh` 报错 `bc: command not found`
- **影响**: 无法计算预算百分比
- **解决**: `brew install bc` (macOS) 或 `apt install bc` (Linux)

### 问题 3: 预算文件不存在
- **现象**: 首次运行时找不到预算文件
- **影响**: 显示默认值 0.0
- **解决**: `restart.sh` 会自动创建，或手动创建
