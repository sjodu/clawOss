## 能否运行 ClawOSS？

### 资源需求分析

**ClawOSS 最低要求：**
- CPU: 1 核（可以运行，但较慢）
- 内存: 1.5GB（最低）
- 硬盘: 10GB（最低）
- 带宽: 1Mbps（足够）

**实际运行资源消耗：**
- Node.js: ~200MB
- OpenClaw Gateway: ~150MB
- Agent 运行时: ~300-500MB
- 系统开销: ~200MB
- **总计: ~850MB - 1GB**

### 结论

| 配置 | 能否运行 | 性能 | 建议 |
|------|---------|------|------|
| 1核1GB | ⚠️ 勉强可以 | 很慢，容易 OOM | 不推荐 |
| 1核2GB | ✅ 可以 | 较慢，但稳定 | 最低推荐 |
| 2核2GB | ✅ 流畅 | 正常 | 推荐 |
| 2核4GB | ✅ 很好 | 快速 | 理想 |

**如果你的服务器是 1核1GB，可以运行，但需要优化配置。**

## 阿里云最低配优化部署方案

### 方案 A：轻量级配置（1核1GB）

#### 优化策略
1. **禁用不必要的服务**
2. **使用 Swap 扩展内存**
3. **限制并发任务**
4. **使用轻量级模型（DeepSeek）**
5. **定期清理日志**

#### 部署脚本

创建 `scripts/deploy-aliyun-minimal.sh`：

```bash
#!/bin/bash
set -e

echo "=========================================="
echo "阿里云最低配服务器部署脚本"
echo "适用于: 1核1GB / 1核2GB"
echo "=========================================="

# 检查系统资源
echo ""
echo "📊 检查系统资源..."
echo "CPU 核心数: $(nproc)"
echo "总内存: $(free -h | awk '/^Mem:/ {print $2}')"
echo "可用内存: $(free -h | awk '/^Mem:/ {print $7}')"
echo "磁盘空间: $(df -h / | awk 'NR==2 {print $4}')"

# 检查内存是否小于 2GB
TOTAL_MEM=$(free -m | awk '/^Mem:/ {print $2}')
if [ $TOTAL_MEM -lt 1800 ]; then
    echo ""
    echo "⚠️  检测到内存不足 2GB，将启用优化配置"
    LOW_MEMORY=true
else
    echo ""
    echo "✅ 内存充足"
    LOW_MEMORY=false
fi

# 1. 创建 Swap（如果内存 < 2GB）
if [ "$LOW_MEMORY" = true ]; then
    echo ""
    echo "📝 步骤 1: 创建 2GB Swap 文件..."
    
    if [ ! -f /swapfile ]; then
        sudo dd if=/dev/zero of=/swapfile bs=1M count=2048 status=progress
        sudo chmod 600 /swapfile
        sudo mkswap /swapfile
        sudo swapon /swapfile
        
        # 永久启用
        if ! grep -q '/swapfile' /etc/fstab; then
            echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
        fi
        
        echo "✅ Swap 创建成功"
    else
        echo "✅ Swap 已存在"
    fi
    
    # 优化 Swap 使用
    sudo sysctl vm.swappiness=10
    echo "vm.swappiness=10" | sudo tee -a /etc/sysctl.conf
fi

# 2. 更新系统
echo ""
echo "📝 步骤 2: 更新系统..."
sudo apt update

# 3. 安装 Node.js（使用 18 而不是 24，更轻量）
echo ""
echo "📝 步骤 3: 安装 Node.js 18..."
if ! command -v node &> /dev/null; then
    curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
    sudo apt install -y nodejs
    echo "✅ Node.js 安装完成: $(node --version)"
else
    echo "✅ Node.js 已安装: $(node --version)"
fi

# 4. 安装 Git
echo ""
echo "📝 步骤 4: 安装 Git..."
if ! command -v git &> /dev/null; then
    sudo apt install -y git
    echo "✅ Git 安装完成"
else
    echo "✅ Git 已安装"
fi

# 5. 安装 GitHub CLI
echo ""
echo "📝 步骤 5: 安装 GitHub CLI..."
if ! command -v gh &> /dev/null; then
    curl -fsSL https://cli.github.com/packages/githubcli-archive-keyring.gpg | sudo dd of=/usr/share/keyrings/githubcli-archive-keyring.gpg
    echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/githubcli-archive-keyring.gpg] https://cli.github.com/packages stable main" | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null
    sudo apt update
    sudo apt install -y gh
    echo "✅ GitHub CLI 安装完成"
else
    echo "✅ GitHub CLI 已安装"
fi

# 6. 安装 OpenClaw（限制内存使用）
echo ""
echo "📝 步骤 6: 安装 OpenClaw..."
if ! command -v openclaw &> /dev/null; then
    # 限制 npm 内存使用
    export NODE_OPTIONS="--max-old-space-size=512"
    sudo npm install -g openclaw --production
    echo "✅ OpenClaw 安装完成"
else
    echo "✅ OpenClaw 已安装"
fi

# 7. 配置环境变量
echo ""
echo "📝 步骤 7: 配置环境变量..."
if [ ! -f .env ]; then
    echo "⚠️  .env 文件不存在，请手动创建"
    echo "参考 .env.example 文件"
    exit 1
fi

# 加载环境变量
source .env

# 8. 配置 GitHub
echo ""
echo "📝 步骤 8: 配置 GitHub..."
git config --global user.name "$GITHUB_USERNAME"
git config --global user.email "$GITHUB_EMAIL"
echo "$GITHUB_TOKEN" | gh auth login --with-token
echo "✅ GitHub 配置完成"

# 9. 创建优化的 OpenClaw 配置
echo ""
echo "📝 步骤 9: 生成优化配置..."

# 创建轻量级配置生成脚本
cat > /tmp/generate_minimal_config.py << 'PYTHON_EOF'
import json
import os

config = {
    "models": {
        "providers": {
            os.getenv("LLM_PROVIDER", "deepseek"): {
                "models": [{
                    "name": os.getenv("LLM_MODEL", "deepseek-chat"),
                    "apiKey": {"$env": "LLM_API_KEY"},
                    "baseURL": os.getenv("LLM_BASE_URL", "https://api.deepseek.com")
                }]
            }
        }
    },
    "gateway": {
        "host": "127.0.0.1",
        "port": 18789,
        "maxConcurrentSessions": 1  # 限制并发
    },
    "agents": {
        "defaults": {
            "maxTokens": 32000,  # 降低 token 限制
            "temperature": 0.7
        }
    },
    "plugins": {
        "entries": []
    }
}

os.makedirs(os.path.expanduser("~/.openclaw"), exist_ok=True)
with open(os.path.expanduser("~/.openclaw/openclaw.json"), "w") as f:
    json.dump(config, f, indent=2)

print("✅ 配置文件已生成")
PYTHON_EOF

python3 /tmp/generate_minimal_config.py

# 10. 初始化预算追踪
echo ""
echo "📝 步骤 10: 初始化预算追踪..."
mkdir -p workspace/memory
echo "${BUDGET_MAX_USD:-5.0}" > workspace/memory/budget-max.txt
echo "0.0" > workspace/memory/budget-spent.txt
echo "✅ 预算追踪初始化完成"

# 11. 创建轻量级启动脚本
echo ""
echo "📝 步骤 11: 创建启动脚本..."

cat > /tmp/start-minimal.sh << 'BASH_EOF'
#!/bin/bash

# 限制 Node.js 内存使用
export NODE_OPTIONS="--max-old-space-size=512"

# 加载环境变量
source .env

# 启动 Gateway
echo "🚀 启动 OpenClaw Gateway..."
openclaw gateway start

# 等待 Gateway 启动
sleep 5

# 检查状态
if openclaw gateway status | grep -q "running"; then
    echo "✅ Gateway 启动成功"
else
    echo "❌ Gateway 启动失败"
    exit 1
fi

echo ""
echo "=========================================="
echo "✅ ClawOSS 启动成功！"
echo "=========================================="
echo ""
echo "监控命令："
echo "  openclaw status          # 查看状态"
echo "  openclaw logs --follow   # 查看日志"
echo "  free -h                  # 查看内存"
echo "  htop                     # 系统监控"
echo ""
BASH_EOF

chmod +x /tmp/start-minimal.sh
sudo mv /tmp/start-minimal.sh /usr/local/bin/clawoss-start

# 12. 创建 systemd 服务（轻量级）
echo ""
echo "📝 步骤 12: 创建系统服务..."

sudo tee /etc/systemd/system/clawoss.service > /dev/null << EOF
[Unit]
Description=ClawOSS Autonomous Agent (Minimal)
After=network.target

[Service]
Type=simple
User=$USER
WorkingDirectory=$(pwd)
Environment="NODE_OPTIONS=--max-old-space-size=512"
Environment="PATH=/usr/local/bin:/usr/bin:/bin"
ExecStart=/usr/local/bin/clawoss-start
Restart=always
RestartSec=30
StandardOutput=append:/var/log/clawoss.log
StandardError=append:/var/log/clawoss-error.log

# 资源限制
MemoryMax=800M
MemoryHigh=700M
CPUQuota=80%

[Install]
WantedBy=multi-user.target
EOF

# 13. 创建日志轮转配置
echo ""
echo "📝 步骤 13: 配置日志轮转..."

sudo tee /etc/logrotate.d/clawoss > /dev/null << EOF
/var/log/clawoss*.log {
    daily
    rotate 3
    compress
    delaycompress
    missingok
    notifempty
    create 0644 $USER $USER
}
EOF

# 14. 创建监控脚本
echo ""
echo "📝 步骤 14: 创建监控脚本..."

cat > /tmp/monitor-minimal.sh << 'BASH_EOF'

echo "=========================================="
echo "ClawOSS 系统监控"
echo "=========================================="
echo ""

# 内存使用
echo "📊 内存使用:"
free -h | grep -E "Mem|Swap"
echo ""

# 磁盘使用
echo "💾 磁盘使用:"
df -h / | tail -1
echo ""

# Gateway 状态
echo "🔧 Gateway 状态:"
openclaw gateway status || echo "Gateway 未运行"
echo ""

# 预算状态
echo "💰 预算状态:"
if [ -f workspace/memory/budget-spent.txt ]; then
    SPENT=$(cat workspace/memory/budget-spent.txt)
    MAX=$(cat workspace/memory/budget-max.txt)
    echo "已使用: \$$SPENT / \$$MAX"
else
    echo "预算文件不存在"
fi
echo ""

# 最近的 PR
echo "📝 最近的 PR:"
gh search prs --author ${GITHUB_USERNAME:-BillionClaw} --limit 3 2>/dev/null || echo "无法获取 PR 信息"
echo ""

echo "=========================================="
BASH_EOF

chmod +x /tmp/monitor-minimal.sh
sudo mv /tmp/monitor-minimal.sh /usr/local/bin/clawoss-monitor

# 15. 启动服务
echo ""
echo "📝 步骤 15: 启动服务..."

sudo systemctl daemon-reload
sudo systemctl enable clawoss
sudo systemctl start clawoss

# 等待启动
sleep 5

# 检查状态
if sudo systemctl is-active --quiet clawoss; then
    echo "✅ 服务启动成功"
else
    echo "❌ 服务启动失败，查看日志:"
    echo "sudo journalctl -u clawoss -n 50"
fi

# 16. 显示完成信息
echo ""
echo "=========================================="
echo "✅ 部署完成！"
echo "=========================================="
echo ""
echo "系统信息:"
echo "  CPU: $(nproc) 核"
echo "  内存: $(free -h | awk '/^Mem:/ {print $2}')"
echo "  Swap: $(free -h | awk '/^Swap:/ {print $2}')"
echo "  磁盘: $(df -h / | awk 'NR==2 {print $4}') 可用"
echo ""
echo "常用命令:"

```

## 性能优化配置

### 1. 限制并发任务

编辑 `~/.openclaw/openclaw.json`：
```json
{
  "gateway": {
    "maxConcurrentSessions": 1
  }
}
```

### 2. 降低心跳频率

编辑 `workspace/HEARTBEAT.md`，将检查间隔从 5 分钟改为 15 分钟。

### 3. 启用内存压缩

```bash
sudo apt install -y zram-config
```

### 4. 定期清理

创建清理脚本 `scripts/cleanup.sh`：
```bash
#!/bin/bash

# 清理 apt 缓存
sudo apt clean
sudo apt autoremove -y

# 清理日志
sudo journalctl --vacuum-time=3d

# 清理 OpenClaw 缓存
rm -rf ~/.openclaw/cache/*

# 清理 npm 缓存
npm cache clean --force

echo "✅ 清理完成"
df -h /
```

设置定时清理：
```bash
# 每周日凌晨 3 点清理
0 3 * * 0 /path/to/scripts/cleanup.sh
```

## 监控和告警

### 1. 内存监控

创建 `scripts/check-memory.sh`：
```bash
#!/bin/bash

USED=$(free | awk '/Mem:/ {printf "%.0f", $3/$2 * 100}')

if [ $USED -gt 90 ]; then
```

```bash
#!/bin/bash

USED=$(df / | awk 'NR==2 {print $5}' | sed 's/%//')

if [ $USED -gt 80 ]; then

    sudo journalctl --vacuum-time=1d
```

## 故障排查

### 问题 1: 内存不足 (OOM)

**症状：** 服务频繁重启，日志显示 "Out of memory"

**解决：**
```bash
# 1. 增加 Swap
sudo swapoff /swapfile
sudo dd if=/dev/zero of=/swapfile bs=1M count=4096
sudo mkswap /swapfile
sudo swapon /swapfile

# 2. 降低内存限制
export NODE_OPTIONS="--max-old-space-size=384"

# 3. 限制并发
# 编辑 ~/.openclaw/openclaw.json
# "maxConcurrentSessions": 1
```

### 问题 2: CPU 使用率过高

**症状：** 服务器响应缓慢

**解决：**
```bash
# 限制 CPU 使用
sudo systemctl set-property clawoss CPUQuota=50%

# 降低心跳频率
# 编辑 workspace/HEARTBEAT.md
# 将间隔改为 15-30 分钟
```

### 问题 3: 磁盘空间不足

**症状：** 无法写入文件

**解决：**
```bash
# 清理日志
sudo journalctl --vacuum-size=100M

# 清理 OpenClaw
rm -rf ~/.openclaw/logs/*

# 清理 apt
sudo apt clean
sudo apt autoremove -y
```
