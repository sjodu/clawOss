## 第一步：安装依赖

### 1. 安装 OpenClaw

```bash
# 使用 npm 全局安装
npm install -g openclaw

# 验证安装
openclaw --version
```

如果没有 npm，先安装 Node.js：
```bash
# macOS
brew install node

# 或下载安装包
# https://nodejs.org/
```

### 2. 安装其他依赖

```bash
# GitHub CLI (如果还没安装)
brew install gh

# jq (JSON 处理工具)
brew install jq

# bc (计算器，用于预算计算)
brew install bc
```

### 3. 验证所有依赖


## 故障排查

### 问题 1: OpenClaw 安装失败

```bash
# 清理旧版本
npm uninstall -g openclaw

# 重新安装
npm install -g openclaw

# 如果还是失败，尝试使用 sudo
sudo npm install -g openclaw
```

### 问题 2: GitHub 认证失败

```bash
# 重新认证
gh auth logout
echo "ghp_your_github_token_here" | gh auth login --with-token

# 验证
gh auth status
gh api user
```

### 问题 3: Gateway 启动失败

```bash
# 查看错误日志
cat ~/.openclaw/logs/gateway.err.log

# 检查端口占用
lsof -i :18789

# 强制停止并重启
pkill -f openclaw
openclaw gateway install
```

### 问题 4: 预算不更新

```bash
# 检查文件权限
ls -la workspace/memory/budget-*.txt

# 手动创建文件
echo "5.0" > workspace/memory/budget-max.txt
echo "0.0" > workspace/memory/budget-spent.txt
chmod 644 workspace/memory/budget-*.txt
```
